Detailed Description

An trimesh shape.

Example:

using namespace cannon;
Body::BodyOptions bodyOptions(world);
bodyOptions.id = world->genBodyId();
bodyOptions.mass = 1;
auto shapeBody = Body::create(bodyOptions);
auto shape = Trimesh::createTorus(world->genShapeId(), 4, 3.5, 16, 16);
shapeBody->addShape(shape);
world->addBody(shapeBody);
