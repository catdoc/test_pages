◆ step()

void World::step ( real_t  dt,
real_t  timeSinceLastCalled = 0.0f,
int  maxSubSteps = 10 
)

Step the physics world forward in time.

There are two modes. The simple mode is fixed timestepping without interpolation. In this case you only use the first argument. The second case uses interpolation. In that you also provide the time since the function was last used, as well as the maximum fixed timesteps to take.

Parameters
dtThe fixed time step size to use.
timeSinceLastCalledThe time elapsed since the function was last called. default is 0
maxSubStepsMaximum number of fixed steps to take per function call. default is 10

Example:

// fixed timestepping without interpolation
world->step(1.0/60.0);
See also
http://bulletphysics.org/mediawiki-1.5.8/index.php/Stepping_The_World
