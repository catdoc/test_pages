void RaycastVehicle::updateWheelTransform ( int  wheelIndex)

Update one of the wheel transform.

when rendering wheels: during each step, wheel transforms are updated BEFORE the chassis; ie. their position becomes invalid after the step. Thus when you render wheels, you must update wheel transforms before rendering them. See raycastVehicle demo for an example.

Parameters
